# Urban 3D Point Cloud Segmentation

A 3D geospatial data case with bundled point data, clustering, bounds, cluster summaries, and plots.

## Data
- `data/` - Bundled inputs used by the notebook without external downloads.

## Workflow
1. Open main.ipynb in the recommended runtime image.
2. Load the local data from the data/ directory and inspect schema or metadata.
3. Run the analysis cells to compute indicators or model diagnostics.
4. Review static plots and interactive HTML/map outputs written to outputs/.
5. Use the printed ranking or summary table as the case conclusion.

## Source Notes
Inspired by PDAL and PyVista point-cloud processing examples with offline data.
